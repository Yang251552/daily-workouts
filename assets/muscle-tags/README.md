# Weekly muscle-tag assets

`png/` contains the production-ready transparent tags used by `fitness-daily.html`.
They are generated from the selected hand-cut recycled-paper reference and named by stable English muscle keys.

## Coverage

## Research basis

- The weekly action list comes from `weekly_plan.md` and its selected source rows in `exercises.db`.
- ACE classifies push-ups under arms, chest and shoulders, and squats under hips and thighs: <https://www.acefitness.org/resources/everyone/exercise-library/>.
- Cat-cow principally mobilizes the spine and activates the core: <https://health.clevelandclinic.org/cat-cow-stretch>.
- Shoulder circles emphasize shoulders and upper back: <https://liftmanual.com/standing-shoulder-circling/>.
- Marching in place primarily uses hip flexors and quadriceps: <https://getfitcraft.com/exercises/marching-in-place>.
